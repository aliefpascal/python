# equations
$$\tag{1}
x_{i+1} = x_i + \Delta x, \ \ \ \ i = 1, \ 2, \ 3, \ \dots, \ n
$$

$$\tag{2}
x_i = x_1 + (i - 1)\Delta x, \ \ \ \ i = 1, \ 2, \ 3, \ \dots, \ n
$$

$$\tag{3}
\Delta x \ge 0
$$

$$\tag{4}
\delta x_i = x_{i+1} - x_i
$$

$$\tag{5}
\Delta x = \overline{\delta x} = \frac{1}{n - 1} \sum_{i=1}^{n-1} \delta x_i
$$
